# Laravel 12 & Vue 3 Task Manager

A full-stack monorepo application featuring a robust backend API powered by Laravel 12 and a reactive, interactive user interface built with Vue 3.

---

## 📁 Repository Structure

* **`backend/`** - Laravel 12 REST API (Authentication, Task Models, and Controllers)
* **`frontend/`** - Vue 3 Single Page Application (UI, State Management, and API integration)
